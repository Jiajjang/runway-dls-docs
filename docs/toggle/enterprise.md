# USAGE GUIDELINES

- Use for settings that can be turned on or off.

- Use when changes apply immediately—toggles should not require a “Save” action.

- Labels must clearly describe the setting being controlled.

- Use subtext only when context requires additional explanation.

- Avoid using toggles for actions (e.g., “Submit”, “Refresh”).

- Place toggle on the right with label on the left for readability.
