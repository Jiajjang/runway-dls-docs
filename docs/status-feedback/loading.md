
# Loading

Loading indicators communicate to users that 
the system is processing, fetching, or waiting for 
something to complete. They help set expectations, 
reduce uncertainty, and maintain trust.

---

# USAGE GUIDELINES

- Always place the loading indicator in the center of the screen or container.

- Ensure it is visually balanced within the available space.

- Avoid placing loaders near edges or corners.

- Use JSON files to configure and control loading states.
