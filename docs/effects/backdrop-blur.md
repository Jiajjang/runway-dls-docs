
# Backdrop Blur

Backdrop blur is used to soften content behind 
a foreground element, allowing background context 
to remain visible while reducing visual prominence. 

---

# USAGE GUIDELINES

Backdrop blur is applied to the background layer behind a foreground surface.

Use backdrop blur when:

- A foreground element needs separation without fully obscuring the background

- Background content should remain visible but de&hyphen;emphasised

- The UI state is temporary or layered
