
# Drop Shadow

Drop Shadows are used to create depth and 
visual separation between surfaces. 

---

# USAGE GUIDELINES

Choose the shadow variant based on the background on which the component is placed.
Shadows are applied to:

- Card

- Floating containers

- Elevated surfaces

**Do not: **

- Mix light and dark shadow variants within the same surface

- Stack multiple shadows on a single element

Shadows are used exclusively to indicate elevation and surface separation.

## Variants
