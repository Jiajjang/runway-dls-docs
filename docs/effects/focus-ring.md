
# Focus Ring

Focus rings are used to indicate the currently 
focused element on a page. They provide a clear 
visual indicator for keyboard and non-pointer 
navigation, supporting accessibility across products.

---

# USAGE GUIDELINES

Apply focus rings to interactive elements when they receive focus.

Focus rings are shown:

- On keyboard focus

- On programmatic focus


Use **single&hyphen;layer focus ring**s for elements that **do not require shadows**, such as:

- Toggle

- Checkboxes

- Simple form controls

> Please note:  
> Do not remove or suppress focus rings unless an alternative focus indicator of equal visibility is provided.
