
# Grid Layout

Grids define the structural framework for 
layouts across CAG digital products. 

---

# GRID SYSTEM

The grid system provides predefined layout structures for common screen sizes and platforms. These grids define column count, gutters, and margins, and should be used as the starting point for all layouts.

The system supports:

- Desktop layout

- Tablet layouts

- Mobile (iPhone) layouts

- Container&hyphen;based grids for modular content

> Some extra info:  
> Grids are available as layout presets in Figma and should not be recreated manually.
