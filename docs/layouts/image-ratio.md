# ASPECT RATIOS

The Design Language System supports a fixed set of 7 aspect ratios. These ratios are designed to work with Adobe Experience Manager (AEM) &hyphen; [Dynamic Media](https://experienceleague.adobe.com/en/docs/experience-manager-learn/assets/dynamic-media/dynamic-media-overview-feature-video-use) and should be used consistently across products.

For more info on Aspect Ratio Streamlining, refer to: [CAG Aspect Ratio Streamlining](https://changiairport.sharepoint.com/:p:/r/sites/CA.comRevamp/Shared%20Documents/13.%20Design/01.%20Dynamic%20Media/CA%20Revamp%20-%20Aspect%20Ratio%20%26%20Component%20Resizing.pptx?d=w231efd016e3d49259a668b6380c5cf9a&csf=1&web=1&e=q3c7II&nav=eyJzSWQiOjIxNDc0NzYwNTAsImNJZCI6NDE3MzUzNjA2MH0)

# DYNAMIC MEDIA

Dynamic Media works by having a master image that would be uploaded to the platform. The image will then automatically be cropped into the aspect ratio sizes using auto / smart detection of the subject matter.

  
**Description**  
![Img](https://studio-assets.supernova.io/design-systems/570165/f03754f2-dbe3-47f6-a148-9ee3258b4a58.png)  
The master image specifications:** **  
**Dimensions: **1920px x 1080px (HD)
**Suggested Resolution:** 150ppi
**Suggested Media Size:** below 5 MB
**Colour Mode:** RGB / sRGB
