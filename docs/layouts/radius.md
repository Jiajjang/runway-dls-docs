
# Radius

Radius defines how rounded the corners of 
UI elements appear across the interface. 

---

# RADIUS SYSTEM

The radius system is based on a limited, predefined set of values. Working from a constrained scale allows teams to design faster while maintaining consistency across products and platforms.

Radius values can be applied to:

- Containers and card

- Buttons and form elements

- Images and media

- Interactive UI surfaces

# Commonly Used Radius

While the full scale is available, certain values are used more frequently to maintain visual consistency across products.

Common patterns:

- Big cards: 16px

- Small / less&hyphen;rounded cards: 4px
