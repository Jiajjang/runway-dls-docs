
# Spacing

Spacing defines how elements are positioned 
and separated across the interface.

---

# USAGE GUIDELINES

- Multiples of **4px **are used as the primary unit to determine spatial dimensions such as spacing, padding, and element sizing.

- Multiples of** 2px** may be used for more precise adjustments.

## Page&hyphen;level Spacing

Page&hyphen;level spacing establishes overall layout structure and separation between major content areas.

Standard page&hyphen;level spacing:

- 24px left and right margin

- iPhones use a 16px left and right margin

- 24px spacing between components on a page

- 40px spacing between major content sections on a mobile page. 80px spacing between major content sections on a mobile page. 

These values should be used consistently to maintain layout rhythm across screens.
