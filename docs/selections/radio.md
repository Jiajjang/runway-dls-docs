
# Radio — Enterprise

**Category:** Selections

# USAGE GUIDELINES

- Use when only a single option can be selected at a time.

- Display radio buttons in a vertical or horizontal list depending on available space.

- Selectable area includes both the circle and label text.

- Use text when additional explanation is needed.

- Use ‘**lg’** size when accompanied text is in font size 16px; ‘**md’ **size when accompanied text is in font size 14px.

- Disable options that are not available.

# BEHAVIOUR (INTERACTIONS)

  

### Selection
- **Tab** enters or exits the radio group.
- **Arrow keys** (↑/↓ or ←/→) move between options in the group.
- **Space** selects the focused option.

### Keyboard Interactions
- **Tab** moves between checkboxes.
- **Space** toggles the current checkbox.
- Focus ring always visible during keyboard interaction.

### Validation & Error Handling
- Error appears only at the group level, not per radio item.
- Error resolves once a valid selection is made.
