
# Enterprise

# USAGE GUIDELINES

- Use when users need to select one or multiple options from a predefined set.

- Best for options that benefit from visual representation (e.g. icons, images).

- Suitable for comparative choices where users need to scan and decide quickly.

- Keep content concise and scannable.

- Ensure clear selected / unselected states for easy recognition.

- Avoid using when there are too many options (use dropdown or list instead).

# BEHAVIOUR (INTERACTIONS)

  

### Selection
- Single/multi- select are possible depending on required configuration.

### Trigger
- Entire card area is clickable.

### Feedback
- Selected state outlined.
