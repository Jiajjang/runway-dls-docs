
# Enterprise

# USAGE GUIDELINES

- Use Attribute Chips only for displaying information, not for selection or actions.

- Use approved attribute categories.

- Keep labels short and descriptive.

- Place chips near titles, cards, or metadata sections to highlight relevant attributes.

- Do not use for filtering or sorting — use Filter Chips instead.

- Avoid mixing Attribute Chips with interactive components.

- Use small for mobile and medium for desktop
