# USAGE GUIDELINES

- Use for sequential or highlight content such as promotions, destinations, or featured visuals.

- Limit total slides to keep browsing manageable (recommended: up to 6).

- Use image previews when users benefit from visual orientation.

- Use minimal layouts (no preview) when the surrounding UI already provides context.

- Avoid placing essential or time-sensitive information inside a carousel.

- Ensure controls and movement are predictable and non-disruptive.
