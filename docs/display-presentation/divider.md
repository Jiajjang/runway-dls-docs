
# Divider

Provides a thin line used to separate distinct sections, 
groups, or blocks of information.

---

# USAGE GUIDELINES

- Use divider to visually structure content and create separation between sections.

- Use sparingly to avoid clutter and maintain clean layouts.

- Place divider between logical content groups (e.g., form sections, list items).

- Apply the correct divider tone depending on the interface mode (Light or Dark).

- Follow consistent spacing rules above and below divider across screens.
