
# QR Codes & Barcodes

Displays a scannable code used for app actions, 
linking, redemptions, or directing users to Changi 
digital experiences.

---

# USAGE GUIDELINES

- Use **QR Codes** when information needs to be scanned using mobile devices or cameras.

- Use **Barcodes** when integrating with physical scanners or operational systems.

- Ensure the code is displayed **clearly with sufficient contrast** between foreground and background.

- Maintain **adequate spacing around the code** (quiet zone) to ensure reliable scanning.

- Avoid placing QR codes or barcodes on **complex or patterned backgrounds**.

- Ensure the encoded data is **correct and tested for scan reliability** before deployment.

- Provide **alternative access methods** when possible (e.g., URL or reference number) for accessibility.

# CONTENT GUIDELINES

- Ensure encoded data is **accurate and up to date**.

- Avoid encoding **excessively long strings** that may affect scan performance.

- When QR codes link to URLs, ensure the destination is **secure and maintained**

- Provide **clear instructions when scanning is required** (e.g., “Scan to access boarding pass”).
