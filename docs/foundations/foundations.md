Colours, Colour plays a critical role in how users understand, navigate, and interact with CAG digital products.  
  
[![Iconography](https://studio-assets.supernova.io/design-systems/570165/0af824fa-1a93-4405-9532-2c05ec5c8c67.png)](./iconography.md)

# EFFECT

  
[![Drop Shadow](https://studio-assets.supernova.io/design-systems/570165/ee85a84e-2f68-4fe3-9480-820efeb42fe2.png)](./effects/drop-shadow.md)  
Drop Shadow, Drop Shadows are used to create depth and visual separation between surfaces.   
  
[![Focus Ring](https://studio-assets.supernova.io/design-systems/570165/b18fc273-cfe5-40f2-8771-9acd9dfd75d0.png)](./effects/focus-ring.md)  
Focus Ring, Focus rings are used to indicate the currently focused element on a page. They provide a clear visual indicator for keyboard and non&hyphen;pointer navigation, supporting accessibility across products.  
  
[![Backdrop Blur](https://studio-assets.supernova.io/design-systems/570165/07a840a5-2dbc-4e8d-a80d-0b4f634bad27.png)](./effects/backdrop-blur.md)  
Backdrop Blur, Backdrop blur is used to soften content behind a foreground element, allowing background context to remain visible while reducing visual prominence.
