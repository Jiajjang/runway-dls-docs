# OVERVIEW

Typography forms the backbone of visual communication across all CAG digital products.  
It establishes hierarchy, improves scannability, and ensures consistency across mobile, web, and enterprise interfaces.

The CAG typographic system is optimised for:

- **Mobile&hyphen;first design **where density and clarity matter most

- **Operational contexts** where users move quickly and need instant comprehension

- **Accessibility** ensuring comfortable reading for a wide range of users

- **Brand expression** providing a calm, clear, and trustworthy tone suitable for airport environments

Our type scale is purposefully minimal, modular, and stable across all platforms, allowing teams to design confidently without reinventing patterns.

---

# BRAND TYPEFACE

## Lato

  
**Image**  
  
Bold: Lato 700 48px  
Light: Lato 300 48px  
Regular: Lato 400 48px  
Black: Lato 900 48px  
  
  
**Words**  
Our system uses Lato as the primary typeface across all digital products due to its clarity, legibility, and neutral tone.

Weights available:  
- **Light (300)** — long&hyphen;form reading, low&hyphen;emphasis content  
- **Regular (400)** — default for most UI text and body copy  
- **Bold (700)** — emphasis, headers, section titles  
- **Black (900)** — high&hyphen;impact promotional messaging, rare use  

Lato provides excellent readability across languages, platforms, and screen sizes.

# USAGE GUIDELINES

| --- | --- |  
| Hierachy | &hyphen; Use H4 for mobile screen titles (primary header).<br>&hyphen; Use H5–H6 for section breaks (never jump from H4 → Body directly).<br>&hyphen; Avoid skipping heading levels unless there is a clear structural reason. |  
| Content Length | &hyphen; Use Subheading (18px) only for short, 1–4 line blocks.<br>&hyphen; Use Body L for desktop reading experiences.<br>&hyphen; Use Body S as the default mobile paragraph style. |  
| Capitalisation | Use ALL CAPS Heading only for short phrases such as swimlane titles or promo labels.<br>Avoid All Caps for<br>&hyphen; Paragraphs<br>&hyphen; Instructions<br>&hyphen; Menu items<br>&hyphen; Error messages |  
| Typographic Emphasis | Use Bold sparingly to highlight key actions or important terms.<br>
