
# Tooltip

# USAGE GUIDELINES

- Use tooltips for quick clarification or definitions, especially for icons, abbreviations, or unclear UI labels.

- Use only for concise, non-essential information.

- Triggered by hover (desktop) or tap (mobile).

- Should not interrupt the user or require action to proceed.

- Only used when the user initiates interest by hovering or tapping.

- Avoid using tooltips on mobile unless paired with tappable icons.

- Do not use tooltips to convey important instructions or warnings.

- Use Light or Dark tooltip depending on the surrounding contrast or background complexity.

- Keep content extremely short (1–2 lines).
